Requerimientos por usuario

Requerimientos de la propietaria

La propietaria debe ser capaz de:

Visualizar su lista de pedidos

  1. ID de pedido.
  2. Nombre del cliente.
  3. Estado de pedido.
  4. Fecha de pedido.
  5. Fecha de entrega.
  6. Monto total.

Acceder a detalles específicos al seleccionar un pedido

  1. Tipos de flores seleccionadas.
  2. Cantidades de cada tipo de flor.
  3. Estado de pedido.
  4. Personalización (si aplica).

Visualizar los datos del cliente

  1. Nombre.
  2. Dirección de entrega.
  3. Teléfono.
  4. Referencia de entrega (si se proporciona).
  5. Mensaje de "De" y "Para" (si aplica).

Acceder a la información de pago

  1. Método de pago Tarjeta (No. tarjeta, Cuenta, Fecha, CVV) o transferencia (Comprobante).
  2. Estado del pago (Completo, Pendiente, Fallido).

Modificar los pedidos

  1. Cambiar el estado (e.g., a "En Proceso" o "Entregado").
  2. Cancelar un pedido (con confirmación).
  3. Imprimir una copia del pedido para archivo.
